About EasyLogged

EasyLogged is a free login controlled system created by AngularJS and based on Firebase. You can use it right now, simply download it on our github repository. The theme is open source, and you can use it for any purpose, personal or commercial.

EasyLogged includes an AngularJS component (easy-modal) and an AngularJS directive.
